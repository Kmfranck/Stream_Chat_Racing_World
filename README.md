# Stream-Chat-Racing---World-Championship-
Un jeu de course interactif conçu pour les streamers (Twitch &amp; YouTube). Les spectateurs participent en temps réel en tapant le nom d'un pays dans le chat pour faire avancer la voiture correspondante. Le projet gère la double diffusion (multistream), offre des animations fluides avec un moteur physique simple basé sur Pygame.
